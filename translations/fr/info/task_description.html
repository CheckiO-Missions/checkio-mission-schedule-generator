Vous disposez d'une séquence de salariés avec leurs préférences de travail et leurs compétences. Vous avez également des besoins professionnels en termes de temps de travail et de tâches. Votre fonction doit renvoyer un planning qui répond aux besoins de l'entreprise avec les employés disponibles (si c'est possible). Entrons maintenant dans les détails.

business_needs est un tableau{% if is_js %}n{% else %} list{% endif %} de trois valeurs: un jour de la semaine pour lequel établir un horaire - chaîne de caractères; le nombre d'équipes à effectuer ce jour - nombre entier; les tâches à effectuer ce jour - {% if is_js %}tableau{% else %}list{% endif %} de chaînes de caractères.

Remarque concernant les équipes. Chaque employé peut travailler à la première équipe, à la deuxième ou aux deux (toute la journée), à l'exception de ses jours de congé. Le nombre d'équipes pour une entreprise signifie le nombre d'équipes différentes présentes ce jour-là. Par exemple, si le nombre d'équipes est de 3, il peut être couvert par un salarié qui travaille toute la journée (ses deux équipes, "première" et "deuxième", au total - 2) et par n'importe quel autre salarié (car chaque salarié a au moins une équipe), ou par trois équipes d'un salarié différent. Le nombre d'équipes doit être réparti de manière égale entre les équipes de jour "first" et "second". Si le nombre d'équipes est impair (et différent de 1), l'équipe de jour "first" doit comprendre une équipe de plus que l'équipe de jour "second".

Les compétences peuvent être partiellement/totalement les mêmes pour différents employés. Par conséquent, lorsque vous remplissez les équipes de jour, choisissez les salariés ayant le moins de compétences - ils sont généralement moins bien payés. Si le nombre de compétences est identique, choisissez-les par ordre alphabétique. Du point de vue de l'engagement des employés, l'équipe du premier jour est prioritaire: il est plus important d'avoir des employés ayant moins de compétences. Les tâches constituent une séquence minimale à accomplir - la séquence des compétences agrégées pour ce jour peut être plus large. Chaque tâche doit être effectuée (couverte par la compétence correspondante) par une personne de l'équipe de jour "first" ou "second".

S'il n'est pas possible d'établir un calendrier, les deux tableaux intérieurs {% if is_js %}{% else %} lists{% endif %} doivent être vides.
